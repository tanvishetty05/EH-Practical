About Bachat Khata

Bachat Khata is a demo banking website that imitates the look and feel of a real online banking portal, including landing page, login flow and a simple dashboard layout.

Purpose of this project

This project is created for UI/UX practice, learning and portfolio demonstration. It helps you understand how a modern digital banking system might present account information, actions and security messages to users.

Not a real bank

Bachat Khata does not accept deposits, process real transactions or store real customer data. All values and account details are fictional and generated purely for design and development practice.

How it could evolve

In a production system, this frontend could be connected to a secure backend built with frameworks like Laravel, with proper authentication, authorization, encrypted APIs and regulatory-compliant KYC flows.
